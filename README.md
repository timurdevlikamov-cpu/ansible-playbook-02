# Ansible Playbook: ClickHouse & Vector Deployment

Репозиторий содержит Ansible-плейбук для автоматизированной установки, настройки и запуска базы данных ClickHouse и агента сбора логов Vector.

## 📋 Описание
Плейбук выполняет следующие действия:
**1. Установка ClickHouse:**
* Загрузка необходимых .deb пакетов (client, server, common-static) напрямую из официального репозитория.
* Установка пакетов через apt.
* Перезапуск сервиса clickhouse-server.
* Автоматическое создание базы данных logs.
**2. Установка и настройка Vector:**
* Создание необходимых директорий для работы приложения.
* Загрузка и распаковка бинарных файлов Vector из официального архива.
* Развертывание конфигурационного файла на основе Jinja2-шаблона.
* Запуск процесса Vector в фоновом режиме.

## 🚀 Использование

### Предварительные требования
* Установленный Ansible на мастер-хосте.
* Доступ по SSH к целевым хостам (настроенные ключи в ~/.ssh/id_rsa).
* Права sudo для пользователя, указанного в инвентаре.

### Запуск
1. Проверка синтаксиса и режима Dry-run:
   
   ```bash
   nansible-playbook -i inventory/prod.yml site.yml --check --diff```
   
3. Полный запуск:
   
   ```bash
   ansible-playbook -i inventory/prod.yml site.yml```
